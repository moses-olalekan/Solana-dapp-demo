
# Solana dApp Demo

A simple decentralized application built on the Solana blockchain using **Rust** and **Anchor**.

## Features
- On-chain data storage and retrieval
- Basic program structure using Anchor
- Example client interaction

## Tech Stack
- Rust

- Solana Web3.js
- TypeScript
